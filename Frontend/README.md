# 🚀 Hartalis-Frontend

**NOTE:** This repository contains only the frontend of Hartalis. The backend repository and API keys are mandatory to run the full application.

---

## 🛠️ Tech Stack
This frontend is built using modern, high-performance web technologies:
- **React 18**
- **TypeScript**
- **Vite**
- **Tailwind CSS (v3)**
- **React Router DOM**

---

## 💻 How to Run Locally

### 1. Start the Frontend (React/Vite)
Open your terminal, navigate to the `z-ai-frontend` (or `hartalis-frontend`) directory, and run the following commands:

```bash
# Install all required dependencies
npm install

# Start the Vite development server
npm run dev
